#### HTTP 请求概述--- 浏览器输入 url 发生了什么

* DNS 解析，建立 TCP 连接， 发送 HTTP 请求

* server 接受到 HTTP 请求，处理，并返回；

* 客户端收到返回数据，渲染在页面上

     



#### nodejs 处理 HTTP 请求

##### get 请求  和 querystring

> 1、get 请求， 客户端向 server端获取数据；
>
> 2、通过 queryString (url) 来传递数据；
>
> 3、浏览器直接访问，就是发送 get 请求；

##### post 请求 和 postData

> 1、 客户端向服务端传送数据
>
> 2、需要使用 接口请求，传递参数
